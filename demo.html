What is semantic CSS?

Semantic CSS styles native HTML elements directly without utility classes. It uses class names for taxonomy (what something is) rather than presentation (how it looks).

How does dark mode work?

Add class="dark" to the <html> element. All color tokens automatically switch to their dark mode values.

Can I use this with a framework?

Yes. It is a single CSS import with no JavaScript dependency. It works with React, Vue, Svelte, plain HTML, or anything else that renders HTML.

App Shell

The .app-shell class creates a full-page layout with sticky header and sidebar. Apply it to <body> or a wrapper <div>:

<body class="app-shell">
  <header>...</header>
  <aside>...</aside>
  <main>...</main>
</body>

The header spans full width and sticks to the top. The sidebar is sticky below the header. Main content scrolls independently. Omit <aside> for a sidebar-free layout.
